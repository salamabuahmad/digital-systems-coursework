# Digital Systems and Computer Architecture Coursework

This repository contains selected projects from a Digital Systems and Computer Structure course.

The projects focus on SystemVerilog hardware design, combinational and sequential logic, finite-state machines, datapath/control separation, arithmetic circuits, and a simple multicycle RISC-V processor model.

## Projects

### S01 - 64-bit ALU Design
A modular 64-bit ALU built from smaller SystemVerilog components, including multiplexers, a full adder/subtractor unit, and a 1-bit ALU slice.

### S02 - Sequential 32-bit Multiplier
A sequential 32-bit multiplier with separate datapath and finite-state control modules.  
The project also includes an optimized fast multiplier implementation and simulation waveforms.

### S03 - Simple Multicycle RISC-V Processor
A simple multicycle RISC-V processor model with separate datapath and control units, instruction/data memory simulation, and an assembly test program.
